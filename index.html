HOW TO USE THE ADMIN PANEL

  1. Click the tiny ADMIN link in the footer.
  2. Enter your admin password (default: taj1997 — change this, see note below).
  3. Upload a jewellery photo, fill in name, category, price, and badge.
  4. Click ADD TO COLLECTION — it appears on the website instantly.
  5. Use MANAGE ITEMS to edit details/photo or delete any item.
  6. Use SITE IMAGES to change the hero background, about photo, gold loan photo, and the 3 featured design photos.
  7. Use ENQUIRIES to see everyone who has contacted you.
🔒 Security notes: Your session auto-locks after 15 minutes of inactivity, and the login locks for 60 seconds after 5 wrong password attempts. All data is stored only in this browser (localStorage) — it will not appear on other devices/browsers. To change the admin password: open the HTML file, run echo -n "yourNewPassword" | sha256sum in a terminal, and replace the ADMIN_PASS_HASH value near the top of the script with the result. For a production deployment with shared data across devices, connect this site to a real backend (e.g. Firebase, Supabase, or a Node.js server) — ask your developer to wire up the same functions (getItems, saveItems, getEnquiries) to an API instead of localStorage.
